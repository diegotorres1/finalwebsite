    • Design

    The project began during the start of senior year. The primary components of the car involve the software systems on offboard. The software systems were built by me. The system uses Haar Cascades to recognize features that resemble that of a stop sign and a traffic light. The different classes of objects recognized elicit a different response in relation to the acceleration and deceleration of the vehicle. The autonomous steering was a portion of the project that was written, and trained but did not have the chance to test the steering model on the car itself. It remained largely independent feature if there was more time given to implement this into the control schema of the vehicle. The control schema involves a manual control mode and a semi-autonomous mode. The manual mode allows the user to change the speed and angle of the front axle through a simple gui. The autonomous mode involves setting an initial speed to the vehicle and allowing it to react to the environment around it, making decisions based upon these initial parameters. Image processing is done completely offboard while the control decisions are made mostly offboard.
